## SYNOPSIS
    ./nodectl machine node-create PROVIDER METRO NAME <flags>
 
## DESCRIPTION
    Create a node.
 
## POSITIONAL ARGUMENTS
    PROVIDER
    METRO
    NAME
 
## FLAGS
    -s, --size=SIZE
        Type: Optional[]
        Default: None
    -i, --image=IMAGE
        Type: Optional[]
        Default: None
    -k, --keyname=KEYNAME
        Type: Optional[]
        Default: None
    -p, --project=PROJECT
        Type: Optional[]
        Default: None
